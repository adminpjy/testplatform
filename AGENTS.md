# Agent Instructions

## Project Identity

- 当前项目是企业 MIS 智能功能测试平台。
- 系统只面向已有企业 MIS 系统的智能化功能测试。
- 第一阶段直接接入真实被测系统，支持配置、连通性检查、登录检查、执行、复盘和规则沉淀。

## Working Rules

- 每次修改前先分析当前目录、现有代码和任务边界。
- 每次修改保持阶段目标清晰，避免引入未要求的复杂业务能力。
- 每次修改后运行 `scripts/check.ps1`。
- 每阶段完成后执行 Git 提交。
- 不要回滚用户或其他协作者已有修改，除非用户明确要求。
- 产物、报告和运行数据应落在约定目录中，并保持可复盘。
