# Prompt06：维护反馈闭环

## 目标

自动修复失败或人工无法处理时，一键提交维护反馈，维护人员可以直接复盘完整现场。

## 反馈工单内容

1. 项目。
2. 用例。
3. 运行记录。
4. 失败步骤。
5. 用户描述。
6. evidence package。
7. 规则命中。
8. LLM 诊断。
9. 尝试过的修复。
10. 预执行结果。
11. 建议处理方式。

## 维护人员操作

1. 标记系统缺陷。
2. 创建规则。
3. 修改用例。
4. 修改提示词。
5. 标记环境问题。
6. 关闭反馈。

## 验收标准

1. 一键反馈不要求用户手动整理日志。
2. 维护人员打开反馈即可看到完整证据。
3. 反馈处理结果能反哺规则和用例。
