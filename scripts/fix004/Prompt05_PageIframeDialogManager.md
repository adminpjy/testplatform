# Prompt05：新页面、iframe、弹窗统一管理

## 目标

统一处理企业系统中点击后打开新窗口、新标签页、iframe、弹窗、确认框的问题。

## 能力

1. 点击前记录页面集合。
2. 点击后检测：
   - 当前页 URL 变化。
   - 新 page。
   - popup。
   - iframe 出现。
   - dialog 出现。
   - download。
3. 自动选择最可能的业务页面。
4. 执行结束后按策略返回来源页面。

## 弹窗处理

1. 识别确认弹窗。
2. 识别选择弹窗。
3. 识别提示弹窗。
4. 高风险确认必须满足规则门禁。

## 验收标准

1. 目标系统新页面打开后，证据截图在新页面。
2. 关闭新页面后能回到门户或列表。
3. iframe 内控件能被定位。
