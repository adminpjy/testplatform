# Prompt01：缺陷模型与失败分类

## 目标

区分自动化问题、数据问题、环境问题、权限问题和被测系统缺陷。

## 分类

1. script_issue
2. rule_issue
3. test_data_issue
4. permission_issue
5. environment_issue
6. system_defect
7. requirement_issue
8. unknown

## 缺陷字段

1. defect_code
2. title
3. description
4. severity
5. priority
6. project_id
7. case_id
8. run_id
9. failure_sample_id
10. reproduce_steps
11. evidence_paths
12. status
13. assignee

## 验收

1. 失败能分类。
2. 系统缺陷能转缺陷候选。
