# Prompt09：fix006 测试与验收

## 测试范围

后端测试：

1. 分页 Schema。
2. 各模块分页接口。
3. 筛选、搜索、排序。
4. 失败工作台分析。
5. 规则/Prompt 配置助手。
6. 项目报告总览聚合。
7. 权限和高风险门禁。
8. 敏感信息脱敏。

前端测试：

1. 分页表格渲染。
2. 筛选条件保留。
3. 失败工作台操作流。
4. 规则向导。
5. Prompt 试运行。
6. 项目报告下钻。
7. 构建通过。

## 命令

```powershell
python -m pytest backend\tests executor\tests -q
npm --prefix frontend run build
```

## 验收标准

1. 全量测试通过。
2. 所有新增分页接口返回统一结构。
3. 规则库、页面知识库、失败样本、人工介入记录具备分页。
4. 失败样本能智能分析并生成方案。
5. 规则和 Prompt 可通过表单/向导/LLM 辅助维护。
6. 项目报告能总览并下钻到用例。
7. 非专业维护人员不需要直接编辑 JSON 即可完成常见维护任务。
