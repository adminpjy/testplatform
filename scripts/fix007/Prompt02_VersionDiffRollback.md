# Prompt02：版本、差异对比与回滚

## 目标

让测试资产像代码一样可版本管理。

## 版本能力

1. 每次编辑生成新版本。
2. 支持版本说明。
3. 支持版本 diff。
4. 支持回滚。
5. 支持查看发布历史。
6. 支持比较当前草稿和已发布版本。

## Diff 对象

1. 用例自然语言目标。
2. DSL。
3. 测试数据。
4. 成功标准。
5. 规则配置。
6. Prompt 内容。
7. 页面知识结构。

## LLM 辅助

1. 解释两个版本差异。
2. 判断差异风险。
3. 生成版本说明。
4. 建议是否发布。

## 验收

1. 任意资产可查看版本历史。
2. 可一键回滚到某版本。
3. 回滚不会丢失历史版本。
